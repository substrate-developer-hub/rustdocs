    I32(i32),

}

Runtime representation of a value.

Wasm code manipulate values of the four basic value types: integers and floating-point (IEEE 754-2008) data of 32 or 64 bit width each, respectively.

There is no distinction between signed and unsigned integer types. Instead, integers are interpreted by respective operations as either unsigned or signed in twoâ€™s complement representation.
